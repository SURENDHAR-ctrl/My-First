November → May

The Hidden Journey...

First meet panni almost oru semester mudinju, now we are in May. But ungaloda name-ah naan kandupudichadhu exact-ah March month-la dhaan. Why ivlo late-na... honestly, enaku yaariyo pinnadi suthradhoo illa online-la stalk pandradhoo suthama pudikadha oru vishyam. Instead of doing all that, naan just daily neenga pora edathula random-ah show up aaguven. It was never planned, just natural-ah cross aagura madhiri. And I think neenga kooda pala idathula adha note pannirupeenga, right? Remember December-la nadandha semester exams? Anga neenga enna paatheenga, naan ungalai paathen. Actually namba rendu perum pakkathu pakkathu class dhaan. Ungaluku eppo eppo exam-nu check panna, junior-ta 2nd year time table vaangi, adha en time table oda compare panni ungalai find panna try pennen. Finally, God's grace-la neenga en nearby class-ke vandheenga. Department-la enaku MSME project partner-ah select aana oru junior girl irundha. She calls me 'Anna'. High level twist enna-na, kadaisila ava moolama dhaan ungaloda name 'Sabitha' nu naan find pennen. This is exactly how I found you.
